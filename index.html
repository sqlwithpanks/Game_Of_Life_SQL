Game of life sql

An elegant and simple implementation of Conway's Game of Life Using SQL.

Welcome to the most elegant solution in SQL for Conway's Game of Life.

This project is implemented in SQL Server using TSQL and Geospatial(Geometry features) to render the Game of Life grid. See also the blog: https://sqlwithpanks.wordpress.com/2016/06/14/conways-game-of-life-in-tsql/

Have Fun

Just download the sql code file and run it in SQL Server Management Studio. Since I am only using a table variable, you need not worry about cleaning up, nothing is preserved or created :)

Learn

Go through the SQL code, think of it as a puzzle to understand and how it works. There is plenty of learning for anyone interested in honing SQL skills as well as Geospatial or just the fundamentals of the same.
